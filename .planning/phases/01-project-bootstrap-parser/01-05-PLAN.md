2. Create `cmd/stc/main.go`:
   - Import `github.com/spf13/cobra`
   - Create root command:
     ```go
     rootCmd := &cobra.Command{
         Use:     "stc",
         Short:   "IEC 61131-3 Structured Text compiler toolchain",
         Version: version.String(),
     }
     ```
   - Add persistent flag: `rootCmd.PersistentFlags().StringP("format", "f", "text", "Output format: text, json")`
   - Register subcommands: `rootCmd.AddCommand(newParseCmd(), newCheckCmd(), newTestCmd(), newEmitCmd(), newLintCmd(), newFmtCmd(), newPpCmd())`
   - `func main()` calls `rootCmd.Execute()`, exits 1 on error

3. Create `cmd/stc/parse.go`:
   - `func newParseCmd() *cobra.Command` returning a Cobra command:
     - Use: "parse [file...]"
     - Short: "Parse ST source files and output AST"
     - Args: cobra.MinimumNArgs(1)
     - RunE function that:
       a. Reads each file from args using `os.ReadFile`
       b. Calls `parser.Parse(filename, string(content))`
       c. Gets format flag value via `cmd.Flags().GetString("format")`
       d. If format == "json":
          - Create output struct: `type ParseOutput struct { File string; AST *ast.SourceFile; Diagnostics []diag.Diagnostic; HasErrors bool }` with json tags
          - Marshal with `json.MarshalIndent`, print to stdout
       e. If format == "text":
          - Print diagnostics to stderr, one per line using `d.String()` format (file:line:col: severity: message)
          - Print summary to stdout: "Parsed N declaration(s), M diagnostic(s)"
       f. If any file has errors, set exit code 1 (but still output partial results)
   - Handle multiple files: output array of results in JSON mode

4. Create `cmd/stc/version.go`:
   - The version is handled by Cobra's built-in `--version` flag set on rootCmd
   - No separate version subcommand needed — `stc --version` works via Cobra

5. Create `cmd/stc/stubs.go` — stub subcommands for future phases:
   - `func newCheckCmd() *cobra.Command` — Use: "check", Short: "Type-check ST source files", RunE: prints "stc check: not yet implemented" to stderr and returns nil
   - `func newTestCmd() *cobra.Command` — Use: "test", Short: "Run ST unit tests"
   - `func newEmitCmd() *cobra.Command` — Use: "emit", Short: "Emit vendor-specific ST"
   - `func newLintCmd() *cobra.Command` — Use: "lint", Short: "Lint ST source files"
   - `func newFmtCmd() *cobra.Command` — Use: "fmt", Short: "Format ST source files"
   - `func newPpCmd() *cobra.Command` — Use: "pp", Short: "Preprocess ST source files"
   - Each stub respects --format flag and outputs JSON `{"error": "not yet implemented"}` when format is json
   - Each stub returns exit code 0 (not an error, just not implemented)
