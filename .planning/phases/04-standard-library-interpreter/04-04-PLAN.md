  <name>Task 1: Standard library FBs (timers, counters, edge, bistable)</name>
  <files>pkg/interp/stdlib_timers.go, pkg/interp/stdlib_timers_test.go, pkg/interp/stdlib_counters.go, pkg/interp/stdlib_counters_test.go, pkg/interp/stdlib_edge.go, pkg/interp/stdlib_edge_test.go, pkg/interp/stdlib_bistable.go, pkg/interp/stdlib_bistable_test.go</files>
  <behavior>
    Timers (TON):
    - IN=TRUE for 500ms (5x100ms ticks): Q stays FALSE until ET>=PT, then Q=TRUE
    - IN=FALSE: Q=FALSE, ET=0 (immediate reset)
    - ET is capped at PT (never exceeds)
    - Short pulse (IN=TRUE for 200ms then FALSE, PT=500ms): Q never fires
    - Re-trigger (IN falls then rises): ET restarts from 0

    Timers (TOF):
    - IN rises: Q=TRUE immediately, ET=0
    - IN falls: ET starts accumulating, Q stays TRUE until ET>=PT
    - IN rises again before PT: ET resets, Q stays TRUE
    - Rapid toggling faster than PT: Q stays TRUE

    Timers (TP):
    - Rising edge on IN: Q=TRUE for exactly PT, then Q=FALSE
    - Changes to IN during active pulse are ignored
    - ET resets only after pulse completes AND IN is FALSE

    Counters (CTU):
    - Rising edge on CU: CV increments. NOT while CU held TRUE.
    - R=TRUE: CV=0, Q=FALSE. Reset takes priority.
    - CV >= PV: Q=TRUE
    - 5 rising edges with PV=5: Q=TRUE

    Counters (CTD):
    - Rising edge on CD: CV decrements
    - LD=TRUE: CV=PV (load). Load takes priority.
    - CV <= 0: Q=TRUE

    Counters (CTUD):
    - R resets to 0, LD loads PV, CU increments, CD decrements
    - Priority: R > LD > CU/CD
    - QU = (CV >= PV), QD = (CV <= 0)

    Edge (R_TRIG):
    - CLK FALSE->TRUE: Q=TRUE for one scan, then Q=FALSE
    - CLK stays TRUE: Q=FALSE after first scan
    - CLK stays FALSE: Q=FALSE

    Edge (F_TRIG):
    - CLK TRUE->FALSE: Q=TRUE for one scan, then Q=FALSE

    Bistable (SR):
    - S1=TRUE, R=FALSE: Q1=TRUE (set)
    - S1=FALSE, R=TRUE: Q1=FALSE (reset)
    - S1=TRUE, R=TRUE: Q1=TRUE (set dominant)

    Bistable (RS):
    - S=TRUE, R1=FALSE: Q1=TRUE
    - S=FALSE, R1=TRUE: Q1=FALSE
    - S=TRUE, R1=TRUE: Q1=FALSE (reset dominant)
  </behavior>
  <action>
    All FB implementations follow the same pattern: Go struct implementing StandardFB interface from plan 02.

    **stdlib_timers.go**:
    - TON struct: IN bool, PT time.Duration, Q bool, ET time.Duration, running bool. Execute(dt): if !IN -> reset (Q=false, ET=0, running=false). If IN and !running -> start (running=true, ET=0). If running -> ET += dt, cap ET at PT, if ET >= PT then Q=true. SetInput handles "IN" (Bool) and "PT" (Time). GetOutput handles "Q" (Bool) and "ET" (Time).
    - TOF struct: IN bool, PT time.Duration, Q bool, ET time.Duration, prevIN bool, timing bool. Execute(dt): Rising edge (IN && !prevIN) -> Q=true, timing=false, ET=0. IN=true -> Q=true. Falling edge (!IN && prevIN) -> start timing. Timing && !IN -> ET += dt, if ET >= PT then Q=false, timing=false. Cap ET at PT. Save prevIN.
    - TP struct: IN bool, PT time.Duration, Q bool, ET time.Duration, active bool, prevIN bool. Execute(dt): Rising edge && !active -> start pulse (active=true, Q=true, ET=0). Active -> ET += dt, if ET >= PT then Q=false, active only ends when IN also false. Cap ET at PT. Save prevIN.

    **stdlib_counters.go**:
    - CTU struct: CU bool, R bool, PV int64, Q bool, CV int64, prevCU bool. Execute: if R -> CV=0. Elif rising edge CU -> CV++. Q = CV >= PV. Save prevCU.
    - CTD struct: CD bool, LD bool, PV int64, Q bool, CV int64, prevCD bool. Execute: if LD -> CV=PV. Elif rising edge CD -> CV--. Q = CV <= 0. Save prevCD.
    - CTUD struct: CU, CD, R, LD bool, PV int64, QU, QD bool, CV int64, prevCU, prevCD bool. Execute: if R -> CV=0. Elif LD -> CV=PV. Else: rising CU -> CV++, rising CD -> CV--. QU = CV >= PV, QD = CV <= 0. Save prevCU, prevCD.

    **stdlib_edge.go**:
    - RTRIG struct: CLK bool, Q bool, prevCLK bool. Execute: Q = CLK && !prevCLK. Save prevCLK.
    - FTRIG struct: CLK bool, Q bool, prevCLK bool. Execute: Q = !CLK && prevCLK. Save prevCLK.

    **stdlib_bistable.go**:
    - SR struct: S1 bool, R bool, Q1 bool. Execute: Q1 = S1 OR (NOT R AND Q1). Set dominant.
    - RS struct: S bool, R1 bool, Q1 bool. Execute: Q1 = NOT R1 AND (S OR Q1). Reset dominant.

    Register all FBs in StdlibFBFactory map: {"TON": func() StandardFB { return &TON{} }, "TOF": ..., "TP": ..., "CTU": ..., "CTD": ..., "CTUD": ..., "R_TRIG": func() StandardFB { return &RTRIG{} }, "F_TRIG": func() StandardFB { return &FTRIG{} }, "SR": ..., "RS": ...}.

    Per STLB-08: All FBs accept time via Execute(dt). No time.Now() anywhere. Microsecond precision via time.Duration.

    Tests: Table-driven with scan-by-scan scenarios. Each test sets inputs, calls Execute(dt), checks outputs. Multiple scans per test to verify state persistence and edge detection.

    2. **CallStmt dispatch (FB calls with named parameters)**: In execStmt's CallStmt case, resolve callee. If it's an FBInstance in the env, for each CallArg where IsOutput=false: evaluate Value and call fbInstance.SetInput(arg.Name). Call fbInstance.Execute(dt). For each CallArg where IsOutput=true: call fbInstance.GetOutput and set the bound variable in env. If callee is a function (not FB), evaluate as CallExpr instead.

    3. **MemberAccessExpr on FB instances**: In evalExpr's MemberAccessExpr case, if Object resolves to a Value with Kind=ValFBInstance, call FBRef.GetOutput(Member.Name).
