    - evalExpr: Literal "'hello'" (LitString) -> Value{Kind: ValString, Str: "hello"}
    - evalExpr: Literal "T#5s" (LitTime) -> Value{Kind: ValTime, Time: 5*time.Second}
    - evalExpr: BinaryExpr(3 + 4) -> Value{Int: 7}
    - evalExpr: BinaryExpr(10 / 3) -> Value{Int: 3} (integer division)
    - evalExpr: BinaryExpr(10.0 / 3.0) -> Value{Real: 3.333...}
    - evalExpr: BinaryExpr(10 MOD 3) -> Value{Int: 1}
    - evalExpr: BinaryExpr(2 ** 3) -> Value{Real: 8.0}
    - evalExpr: BinaryExpr(5 > 3) -> Value{Bool: true}
    - evalExpr: BinaryExpr(TRUE AND FALSE) -> Value{Bool: false}
    - evalExpr: UnaryExpr(NOT TRUE) -> Value{Bool: false}
    - evalExpr: UnaryExpr(-5) -> Value{Int: -5}
    - evalExpr: Ident("X") with env.Define("X", 10) -> Value{Int: 10}
    - execStmt: AssignStmt(x := 5) sets env X to 5
    - execStmt: IfStmt(TRUE, then=[assign x:=1], else=[assign x:=2]) sets x to 1
    - execStmt: IfStmt(FALSE, then=[assign x:=1], else=[assign x:=2]) sets x to 2
    - execStmt: ForStmt(i := 1 TO 5, body=[sum := sum + i]) results in sum=15
    - execStmt: WhileStmt(x < 5, body=[x := x + 1]) results in x=5
    - execStmt: RepeatStmt(body=[x := x + 1], until x >= 5) results in x=5
    - execStmt: ForStmt with EXIT exits loop early
    - execStmt: ForStmt with CONTINUE skips to next iteration
    - Division by zero returns RuntimeError
  </behavior>
  <action>
    Implement the Interpreter struct with evalExpr and execStatements methods:

    **interpreter.go**: Interpreter struct (minimal for now -- will get extended in plan 02 with FB dispatch). Methods:
    - `evalExpr(env *Env, expr ast.Expr) (Value, error)` -- type-switch on all ast.Expr variants: *ast.Literal (parse int/real/bool/string/time from Value string field using strconv and time parsing), *ast.Ident (env.Get), *ast.BinaryExpr (dispatch on Op.Text: "+", "-", "*", "/", "MOD", "**", "=", "<>", "<", ">", "<=", ">=", "AND", "OR", "XOR" -- handle both int and real operands), *ast.UnaryExpr (NOT, -), *ast.ParenExpr (recurse on Inner), *ast.IndexExpr (array indexing), *ast.CallExpr (stub for now -- will be wired in plan 02/05). Return RuntimeError for unsupported nodes.
    - `evalBinary(env *Env, e *ast.BinaryExpr) (Value, error)` -- evaluate left and right, then dispatch on operator. For arithmetic: if both int, do int arithmetic; if either is real, promote to real. Division by zero returns ErrDivisionByZero. MOD only for ints. Power (**) always returns real via math.Pow.
    - `evalLiteral(lit *ast.Literal) (Value, error)` -- parse Value string into typed Value. LitInt: strconv.ParseInt (handle hex 16#FF, octal 8#77, binary 2#1010 prefixes). LitReal: strconv.ParseFloat. LitBool: strings.EqualFold "TRUE". LitString: strip quotes. LitTime: parse T#5s, T#100ms, T#1h2m3s format into time.Duration. LitTyped: split on # and parse.
    - `execStatements(env *Env, stmts []ast.Statement) error` -- iterate statements, dispatch via execStmt.
    - `execStmt(env *Env, stmt ast.Statement) error` -- type-switch: *ast.AssignStmt (eval value, set in env via target ident), *ast.IfStmt (eval condition, exec then/elsif/else), *ast.CaseStmt (eval expr, match branches), *ast.ForStmt (eval from/to/by, loop with env variable), *ast.WhileStmt (loop while condition true), *ast.RepeatStmt (loop until condition true), *ast.ReturnStmt (return ErrReturn), *ast.ExitStmt (return ErrExit), *ast.ContinueStmt (return ErrContinue), *ast.EmptyStmt (no-op), *ast.CallStmt (stub). Handle ErrExit/ErrContinue in loop bodies.

    Use Op.Text (string) for operator dispatch, not Op.Kind (int), since the text is human-readable ("AND", "+", etc.) and the token kind constants are in the lexer package.

    For time literal parsing, support: T#NNd, T#NNh, T#NNm, T#NNs, T#NNms compound format. Use regex or manual parsing.

    For CaseStmt: evaluate the selector expression once, then for each CaseBranch, evaluate each CaseLabel (CaseLabelValue or CaseLabelRange). For ranges, check low <= selector <= high.

    All tests should construct AST nodes directly (no parsing) to test evaluation in isolation.
