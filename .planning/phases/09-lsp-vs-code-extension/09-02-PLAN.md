<task type="auto">
  <name>Task 1: Position-based symbol lookup and reference finding</name>
  <files>pkg/lsp/navigate.go, pkg/lsp/navigate_test.go</files>
  <action>
1. Create `pkg/lsp/navigate.go` with shared navigation utilities:

   a. `func findIdentAtPosition(file *ast.SourceFile, line, col int) *ast.Ident`:
      - Walk the AST recursively to find the ast.Ident node whose Span contains the given 1-based line:col position
      - Use a recursive visitor that checks each node's Span: node.Span().Start.Line <= line <= node.Span().End.Line (and col within range on boundary lines)
      - When an Ident is found at position, return it
      - Return nil if no ident at position

   b. `func findSymbolAtPosition(doc *Document, line, col int) *symbols.Symbol`:
      - Call findIdentAtPosition to get the ident name
      - If doc.AnalysisResult != nil and doc.AnalysisResult.Symbols != nil, search for the symbol:
        - First try GlobalScope().Lookup(ident.Name)
        - Then iterate all POU scopes: for each child of GlobalScope(), try child.Lookup(ident.Name)
      - Return the matching symbol or nil

   c. `func findAllReferences(doc *Document, name string) []source.Pos`:
      - Walk the AST recursively collecting all ast.Ident nodes where strings.EqualFold(ident.Name, name)
      - Return their Span().Start positions (as source.Pos)

   d. `func collectAllSymbols(table *symbols.Table) []*symbols.Symbol`:
      - Recursively collect symbols from GlobalScope and all children
      - Used by completion handler

   e. `func symbolTypeString(sym *symbols.Symbol) string`:
      - If sym.Type != nil, type-assert to types.Type and call String()
      - Prepend sym.Kind.String() (e.g., "Variable: DINT", "FunctionBlock: FB_Motor")
      - Used by hover handler

2. Create `pkg/lsp/navigate_test.go`:
   - Parse a small ST program with parser.Parse, run analyzer.Analyze
   - Test findIdentAtPosition finds the correct ident at known positions
   - Test findSymbolAtPosition returns the symbol with correct Kind and Type
   - Test findAllReferences finds all occurrences of a variable name
   - Test collectAllSymbols returns symbols from global and POU scopes

  <action>
1. Create `pkg/lsp/definition.go`:
   - `func handleDefinition(store *DocumentStore) protocol.TextDocumentDefinitionFunc`
   - Closure: get doc, call findSymbolAtPosition(doc, params.Position.Line+1, params.Position.Character+1) (convert 0-based LSP to 1-based stc)
   - If symbol found, return []protocol.Location with one entry: URI from symbol.Pos.File (prepend "file://"), Range from symbol.Pos (converting back to 0-based)
   - Return nil if no symbol found (client shows "no definition found")

2. Create `pkg/lsp/hover.go`:
   - `func handleHover(store *DocumentStore) protocol.TextDocumentHoverFunc`
   - Closure: get doc, call findSymbolAtPosition
   - If symbol found, create hover with MarkupContent (kind: "markdown"):
     ```
     **{Kind}** `{Name}` : `{TypeString}`
     ```
   - Return nil if no symbol

3. Create `pkg/lsp/completion.go`:
   - `func handleCompletion(store *DocumentStore) protocol.TextDocumentCompletionFunc`
   - Closure: build completion list from three sources:
     a. IEC 61131-3 keywords: IF, THEN, ELSE, ELSIF, END_IF, FOR, TO, BY, DO, END_FOR, WHILE, END_WHILE, REPEAT, UNTIL, END_REPEAT, CASE, OF, END_CASE, RETURN, EXIT, CONTINUE, VAR, VAR_INPUT, VAR_OUTPUT, VAR_IN_OUT, VAR_TEMP, VAR_GLOBAL, END_VAR, PROGRAM, END_PROGRAM, FUNCTION_BLOCK, END_FUNCTION_BLOCK, FUNCTION, END_FUNCTION, METHOD, END_METHOD, PROPERTY, END_PROPERTY, INTERFACE, END_INTERFACE, TYPE, END_TYPE, STRUCT, END_STRUCT, TRUE, FALSE, NOT, AND, OR, XOR, MOD
        - CompletionItemKind = Keyword (14)
     b. IEC primitive types: BOOL, BYTE, WORD, DWORD, LWORD, SINT, INT, DINT, LINT, USINT, UINT, UDINT, ULINT, REAL, LREAL, STRING, WSTRING, TIME, DATE, DT, TOD
        - CompletionItemKind = TypeParameter (25)
     c. Declared symbols from doc.AnalysisResult.Symbols via collectAllSymbols:
        - Map SymbolKind to CompletionItemKind: Variable->Variable(6), Function->Function(3), FunctionBlock->Class(7), Program->Module(9), Type->Struct(22), EnumValue->EnumMember(20), Interface->Interface(8), Method->Method(2), Property->Property(10)
        - Set Detail to symbolTypeString for type info
   - Return &protocol.CompletionList{IsIncomplete: false, Items: items}

4. Create `pkg/lsp/references.go`:
   - `func handleReferences(store *DocumentStore) protocol.TextDocumentReferencesFunc`
   - Closure: get doc, find ident at position to get the name, call findAllReferences(doc, name)
   - Convert each source.Pos to protocol.Location (0-based conversion)
   - Return locations slice

5. Create `pkg/lsp/rename.go`:
   - `func handleRename(store *DocumentStore) protocol.TextDocumentRenameFunc`
   - Closure: get doc, find ident at position to get old name, call findAllReferences(doc, oldName)
   - Build a WorkspaceEdit with Changes map: one entry per URI, containing TextEdits that replace each occurrence's range with params.NewName
   - For each reference position, the range is: start=pos, end=pos+len(oldName) (all on same line, col+len(oldName))

6. Update `pkg/lsp/server.go`:
   - In the Handler initialization, add these fields:

     - DefinitionProvider: true
     - HoverProvider: true
     - CompletionProvider: &protocol.CompletionOptions{} (empty options = basic completion)
     - ReferencesProvider: true
     - RenameProvider: true
