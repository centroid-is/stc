  <files>pkg/lsp/server.go, pkg/lsp/document.go, pkg/lsp/diagnostics.go, pkg/lsp/formatting.go, pkg/lsp/server_test.go, go.mod, go.sum</files>
  <action>
1. Add GLSP dependency: `go get github.com/tliron/glsp@latest`

2. Create `pkg/lsp/document.go` — in-memory document store:
   - `type Document struct` with fields: URI string, Content string, Version int32, ParseResult *parser.ParseResult, AnalysisResult *analyzer.AnalysisResult
   - `type DocumentStore struct` with a `sync.RWMutex`-protected `map[string]*Document` (keyed by URI)
   - Methods: `Open(uri, content string, version int32)`, `Update(uri, content string, version int32)`, `Close(uri string)`, `Get(uri string) *Document`
   - On Open/Update: call `parser.Parse(uriToFilename(uri), content)` then `analyzer.Analyze([]*ast.SourceFile{parseResult.File}, nil)` storing both results on the Document
   - Helper `uriToFilename(uri string) string` that strips `file://` prefix (use `net/url` package to parse URI)

3. Create `pkg/lsp/diagnostics.go` — diagnostic conversion:
   - `func convertDiagnostics(diags []diag.Diagnostic) []protocol.Diagnostic` — maps each diag.Diagnostic to protocol.Diagnostic
   - Position mapping: diag.Pos.Line-1 and diag.Pos.Col-1 (LSP is 0-based, stc is 1-based)
   - Severity mapping: diag.Error -> protocol.DiagnosticSeverityError (1), diag.Warning -> protocol.DiagnosticSeverityWarning (2), diag.Info -> protocol.DiagnosticSeverityInformation (3), diag.Hint -> protocol.DiagnosticSeverityHint (4)
   - Set Source field to "stc" on every diagnostic
   - `func publishDiagnostics(ctx *glsp.Context, uri string, doc *Document)` — calls convertDiagnostics on combined parse+analysis diags, sends via ctx.Notify with "textDocument/publishDiagnostics"

4. Create `pkg/lsp/formatting.go`:
   - `func handleFormatting(store *DocumentStore) protocol.TextDocumentFormattingFunc` — returns a closure
   - The closure: gets document from store by params.TextDocument.URI, if doc has ParseResult.File, call `format.Format(doc.ParseResult.File, format.DefaultFormatOptions())`
   - Return a single TextEdit replacing the entire document (Range from 0,0 to end-of-document)
   - Count lines in original content to compute the end position

5. Create `pkg/lsp/server.go` — server setup:
   - `func NewServer() *server.Server` that:
     a. Creates a DocumentStore
     b. Creates a protocol.Handler with these fields set:
        - Initialize: return ServerCapabilities with TextDocumentSync=protocol.TextDocumentSyncKindFull (value 1), DocumentFormattingProvider=true
        - Initialized: no-op
        - Shutdown: no-op
        - TextDocumentDidOpen: call store.Open, then publishDiagnostics
        - TextDocumentDidChange: call store.Update with full content (ContentChanges[0].Text since we use Full sync), then publishDiagnostics
        - TextDocumentDidClose: call store.Close, publish empty diagnostics to clear
        - TextDocumentFormatting: delegate to handleFormatting(store)
     c. Returns server.NewServer(&handler, "stc-lsp", false)
   - `func Run()` that calls NewServer().RunStdio()

6. Create `pkg/lsp/server_test.go`:
   - Test `convertDiagnostics` maps severity and positions correctly (1-based to 0-based)
   - Test DocumentStore Open/Update/Close/Get lifecycle
   - Test that Open triggers parse and analysis (check doc.ParseResult != nil, doc.AnalysisResult != nil)
   - Test formatting handler returns a text edit with formatted content
