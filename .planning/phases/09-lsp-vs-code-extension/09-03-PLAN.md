  <name>Task 1: Semantic tokens for inactive preprocessor regions</name>
  <files>pkg/lsp/semantic_tokens.go, pkg/lsp/semantic_tokens_test.go, pkg/lsp/server.go</files>
  <action>
1. Create `pkg/lsp/semantic_tokens.go`:

   a. Define the semantic token legend:
      - Token types: ["comment"] (index 0) — we use "comment" type for inactive regions so editors gray them out
      - Token modifiers: [] (empty)
      - `var semanticTokensLegend = protocol.SemanticTokensLegend{TokenTypes: []string{"comment"}, TokenModifiers: []string{}}`

   b. `func findInactiveRegions(content string) []inactiveRegion`:
      - Scan content line by line
      - Track preprocessor directive stack: when hitting `{IF ...}`, `{ELSIF ...}`, `{ELSE}`, `{END_IF}`:
        - Use a simple stack of booleans indicating whether current branch is "potentially active"
        - Since we do not know the defines at LSP time (no --define flag), mark ALL branches inside preprocessor blocks as "conditional" using a heuristic:
          - Lines between `{IF ...}` and the matching `{END_IF}` that are NOT the directive lines themselves could be in inactive branches
          - Simpler approach: just mark the directive lines themselves ({IF...}, {ELSIF...}, {ELSE}, {END_IF}) as comment-styled semantic tokens, giving them a visual distinction
          - BETTER approach for "grayed out": without knowing active defines, gray out everything inside {ELSE} and {ELSIF} blocks (assume the first {IF} branch is the "default active" one)
      - Return []inactiveRegion where each has startLine, startCol, endLine, endCol (0-based for LSP)

   c. `func handleSemanticTokensFull(store *DocumentStore) protocol.TextDocumentSemanticTokensFullFunc`:
      - Closure: get document content, call findInactiveRegions
      - Encode tokens into the delta-encoded uint32 array per LSP spec:
        - For each inactive line in a region, emit one token per line:
          [deltaLine, deltaStartChar, lineLength, 0 (comment type), 0 (no modifiers)]
        - deltaLine = currentLine - previousLine, deltaStartChar = 0 (or col if first)
      - Return &protocol.SemanticTokens{Data: data}

2. Update `pkg/lsp/server.go`:
   - In Initialize handler, add SemanticTokensProvider:
     ```go
     SemanticTokensProvider: &protocol.SemanticTokensOptions{
         Legend: semanticTokensLegend,
         Full:   true,
     }
     ```
   - Add handler: TextDocumentSemanticTokensFull: handleSemanticTokensFull(store)

3. Create `pkg/lsp/semantic_tokens_test.go`:
   - Test findInactiveRegions with input containing:
     ```
     PROGRAM Main
     {IF defined(VENDOR_BECKHOFF)}
     x := 1;
     {ELSE}
     x := 2;
     {END_IF}
     END_PROGRAM
     ```
   - Assert the {ELSE} block (line with "x := 2;") is marked as inactive
   - Test encoding produces correct delta-encoded uint32 array
   - Test empty input returns empty tokens
   - Test input with no preprocessor directives returns empty tokens
