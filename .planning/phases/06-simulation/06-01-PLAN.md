  <name>Task 1: WaveformGenerator with Step, Ramp, Sine, Square patterns</name>
  <files>pkg/sim/waveform.go, pkg/sim/waveform_test.go</files>
  <behavior>
    - Step(t=0, delay=1s) -> 0.0; Step(t=1.5s, delay=1s) -> amplitude
    - Ramp(t=0) -> offset; Ramp(t=period) -> offset+amplitude
    - Sine(t=0) -> offset; Sine(t=period/4) -> offset+amplitude (peak)
    - Square(t=0, period=2s) -> amplitude; Square(t=1s, period=2s) -> 0 (or low)
    - All return interp.RealValue
    - Config: amplitude, frequency (Hz), offset, delay (for step), duty_cycle (for square, default 0.5)
    - Generate(t time.Duration) returns interp.Value
    - Deterministic: same t -> same output always
  </behavior>
  <action>
    Create `pkg/sim/waveform.go`:

    Define `WaveformKind` as iota const: `WaveStep`, `WaveRamp`, `WaveSine`, `WaveSquare`.

    Define `WaveformConfig` struct:
    - Kind WaveformKind
    - Amplitude float64 (default 1.0)
    - Frequency float64 (Hz, default 1.0 — period = 1/frequency)
    - Offset float64 (default 0.0)
    - Delay time.Duration (for Step: time before stepping, default 0)
    - DutyCycle float64 (for Square: fraction of period that is high, default 0.5)

    Define `WaveformGenerator` struct holding config.

    `NewWaveformGenerator(cfg WaveformConfig) *WaveformGenerator` — constructor, apply defaults (amplitude=1 if 0, frequency=1 if 0, dutyCycle=0.5 if 0).

    `func (w *WaveformGenerator) Generate(t time.Duration) interp.Value` — pure function of t:
    - Step: if t < delay return RealValue(offset), else return RealValue(offset + amplitude)
    - Ramp: period = 1/frequency. phase = t % period. return RealValue(offset + amplitude * (phase/period)). Clamp to 1 period if frequency is 0.
    - Sine: return RealValue(offset + amplitude * math.Sin(2*pi*frequency*t.Seconds()))
    - Square: period = 1/frequency. phase = t % period. if phase < dutyCycle*period return RealValue(offset + amplitude), else return RealValue(offset)

    All math uses `t.Seconds()` for float64 conversion. No wall-clock anywhere.
  </action>

  </behavior>
  <action>
    Create `pkg/sim/plant.go`:

    Define `PlantModel` interface:
    ```go
    type PlantModel interface {
        Update(inputs map[string]interp.Value, dt time.Duration) map[string]interp.Value
    }
    ```

    Input/output keys are UPPERCASE strings (matching ScanCycleEngine convention).

    **MotorModel** struct:
    - Config: MaxSpeed float64 (default 1500 RPM), RampTime time.Duration (time to reach max speed, default 2s)
    - Internal state: currentSpeed float64
    - Update: if input "RUN" is truthy, accelerate toward MaxSpeed at rate MaxSpeed/RampTime * dt. If not, decelerate at same rate toward 0. Clamp to [0, MaxSpeed].
    - Outputs: "SPEED" = RealValue(currentSpeed), "AT_SPEED" = BoolValue(abs(currentSpeed - MaxSpeed) < 0.01 * MaxSpeed) when running

    **ValveModel** struct:
    - Config: TravelTime time.Duration (time for full open/close, default 1s), MaxFlow float64 (default 100.0)
    - Internal state: position float64 (0.0 = closed, 1.0 = fully open)
    - Update: if input "OPEN" is truthy, increase position by dt/TravelTime. If not, decrease by dt/TravelTime. Clamp [0, 1].
    - Outputs: "POSITION" = RealValue(position), "FLOW" = RealValue(position * MaxFlow)

    **CylinderModel** struct:
    - Config: Stroke float64 (default 1.0 meters), Speed float64 (m/s, default 0.1)
    - Internal state: position float64
    - Update: if input "EXTEND" is truthy and not "RETRACT", move position += Speed * dt.Seconds(). If "RETRACT" and not "EXTEND", move position -= Speed * dt.Seconds(). Clamp [0, Stroke].
    - Outputs: "POSITION" = RealValue(position), "AT_EXTEND" = BoolValue(position >= Stroke), "AT_RETRACT" = BoolValue(position <= 0)

    Constructor functions: `NewMotorModel(cfg MotorConfig)`, `NewValveModel(cfg ValveConfig)`, `NewCylinderModel(cfg CylinderConfig)`. Apply defaults in constructors (if MaxSpeed==0 set 1500, etc.).

    Helper: `boolFromValue(v interp.Value) bool` — returns v.IsTruthy(). Helper: `getInput(inputs map[string]interp.Value, key string) interp.Value` — case-insensitive lookup via strings.ToUpper.
