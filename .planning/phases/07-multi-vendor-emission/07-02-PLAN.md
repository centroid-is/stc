Create `cmd/stc/emit_cmd.go`:
- `newEmitCmd() *cobra.Command` with Use "emit [file...]", Short "Emit vendor-specific ST"
- Add `--target` flag (string, default "portable") — valid values: beckhoff, schneider, portable
- `runEmit(cmd *cobra.Command, args []string) error` — the handler:
  1. Validate at least 1 file argument (print usage hint and exit 1 if none)
  2. Read --target and --format flags
  3. For each file: read content with os.ReadFile, parse with parser.Parse, check for parse errors
  4. If parse errors: print diagnostics, set exit code 1, continue to next file
  5. Call emit.Emit(result.File, opts) to get emitted ST text
  6. Text format: print emitted ST to stdout. If multiple files, print `// --- file: {name} ---` separator between files.
  7. JSON format: collect into struct with fields `file string`, `code string`, `diagnostics []diag.Diagnostic`, `has_errors bool`. Marshal as JSON array (or single object for 1 file, matching parse.go pattern).

JSON output structure:
```go
type emitOutput struct {
    File        string            `json:"file"`
    Code        string            `json:"code"`
    Target      string            `json:"target"`
    Diagnostics []diag.Diagnostic `json:"diagnostics"`
    HasErrors   bool              `json:"has_errors"`
}
```
