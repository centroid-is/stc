    ```go
    // AssertionResult records one assertion outcome.
    type AssertionResult struct {
        Passed  bool
        Message string
        Pos     ast.Pos
    }

    // AssertionCollector gathers assertion results during test execution.
    // It is NOT a global -- each test case gets its own collector.
    type AssertionCollector struct {
        Results []AssertionResult
    }

    func (c *AssertionCollector) Record(passed bool, msg string, pos ast.Pos) {
        c.Results = append(c.Results, AssertionResult{Passed: passed, Message: msg, Pos: pos})
    }

    func (c *AssertionCollector) HasFailures() bool { /* check Results */ }
    func (c *AssertionCollector) Failures() []AssertionResult { /* filter !Passed */ }
    ```

    2. **Assertion dispatch in interpreter**: Add a `Collector *AssertionCollector` field and a `LocalFunctions map[string]func(args []Value) (Value, error)` field to the `Interpreter` struct. This solves the StdlibFunctions mutation pitfall -- test-specific functions (ADVANCE_TIME, assertions) live on the interpreter instance, not the global map.

    3. **Modify `evalCall` in pkg/interp/interpreter.go**: BEFORE checking StdlibFunctions, check `interp.LocalFunctions[calleeName]`. This gives per-interpreter-instance function overrides priority over the global map. This is the key change that enables test isolation.

The source position must come from the CallExpr -- modify evalCall to pass `e.Span().Start` to LocalFunctions via a closure capture or by extending the function signature. **Recommended approach**: Make LocalFunctions type `map[string]func(args []Value, pos ast.Pos) (Value, error)` (position-aware signature). In evalCall, when calling a LocalFunction, pass `e.Span().Start`. When calling StdlibFunctions (existing), pass without position.
    - `ASSERT_FALSE(expr [, msg])`: Check `!args[0].IsTruthy()`
    - `ASSERT_EQ(actual, expected [, msg])`: Use `valuesEqual(args[0], args[1])`
    - `ASSERT_NEAR(actual, expected, epsilon [, msg])`: Use `math.Abs(toFloat(a)-toFloat(b)) <= toFloat(eps)`
