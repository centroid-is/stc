      ```go
      type ppOutput struct {
          File        string            `json:"file"`
          Output      string            `json:"output"`
          SourceMap   []smEntry         `json:"source_map"`
          Diagnostics []diag.Diagnostic `json:"diagnostics"`
          HasErrors   bool              `json:"has_errors"`
      }
      type smEntry struct {
          PreprocLine int    `json:"preproc_line"`
          OrigFile    string `json:"orig_file"`
          OrigLine    int    `json:"orig_line"`
      }
      ```
    - Function newPpCmd() *cobra.Command:
      - Use: "pp [file...]"
      - Short: "Preprocess ST source files"
      - Long: explains conditional compilation and --define
      - Args: cobra.MinimumNArgs(1)
      - Add --define/-D flag as StringSlice (supports multiple: --define X --define Y)
      - RunE: runPp
    - Function runPp(cmd *cobra.Command, args []string) error:
      - Parse --define flags into map[string]bool
      - Parse --format flag (inherited from root)
      - For each file arg:
        - Read file content
        - Call preprocess.Preprocess(content, Options{Filename: file, Defines: defines})
        - Text mode: print preprocessed output to stdout, diagnostics to stderr
        - JSON mode: collect ppOutput structs
      - JSON mode: marshal and print (single object for 1 file, array for multiple)
      - If any diagnostic is Error severity, exit with code 1 (same pattern as parse.go)

    Create testdata/preprocess/vendor_portable.st:
    ```
    PROGRAM Main
    VAR
        x : INT;
    END_VAR
    {IF defined(VENDOR_BECKHOFF)}
        x := 1; (* Beckhoff path *)
    {ELSIF defined(VENDOR_SCHNEIDER)}
        x := 2; (* Schneider path *)
    {ELSE}
        x := 0; (* Generic path *)
    {END_IF}
    END_PROGRAM
    ```

    Create testdata/preprocess/define_local.st:

    ```
    {END_IF}
    END_PROGRAM
    ```
