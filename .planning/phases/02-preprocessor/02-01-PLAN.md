    - Preprocess with {IF defined(X)} where X is defined includes the block
    - Preprocess with {IF defined(X)} where X is NOT defined excludes the block
    - Preprocess with {IF}/{ELSIF}/{ELSE}/{END_IF} selects correct branch
    - Preprocess with nested {IF} blocks evaluates correctly
    - Preprocess with {DEFINE MY_FLAG} makes MY_FLAG available for subsequent {IF defined(MY_FLAG)}
    - Preprocess with {ERROR "msg"} in active branch returns error diagnostic with original line:col
    - Preprocess with {ERROR "msg"} in inactive branch does NOT emit error
    - Preprocess preserves non-directive pragmas like {attribute '...'} in output
    - Unmatched {END_IF} produces error diagnostic
    - Missing {END_IF} at end of file produces error diagnostic
    - Source map maps preprocessed line 1 back to original line 1 when no directives removed lines
    - Source map maps correctly when lines are removed by inactive branches
    - Result.Output contains only active lines with directives stripped
    - Result.Diags contains any errors (unmatched directives, {ERROR} hits)
  </behavior>
  <action>
    Create pkg/preprocess/preprocess.go:
    - Define Result struct: Output string, SourceMap *SourceMap, Diags []diag.Diagnostic
    - Define Options struct: Filename string, Defines map[string]bool (external defines from CLI)
    - Function Preprocess(source string, opts Options) Result
    - Implementation approach: line-by-line scan of source text
      1. Split source into lines
      2. Maintain a stack of (active bool, branchTaken bool) for nested IF blocks
      3. For each line, check if it starts with a pragma by looking for "{" after optional whitespace
      4. If pragma found, extract full pragma text (may span to "}" on same line), call parseDirective
      5. If parseDirective returns nil (not a preprocessor directive), treat as normal line
      6. For IF directive: push (evalCondition(cond, defines), false) onto stack; if active, set branchTaken=true
      7. For ELSIF: if no branch taken yet and condition true, activate; otherwise deactivate
      8. For ELSE: if no branch taken yet, activate
      9. For END_IF: pop stack; error if stack empty
      10. For DEFINE: if currently active, add name to defines map
      11. For ERROR: if currently active, add error diagnostic with original position
      12. For non-directive lines: if all stack entries are active, emit line to output and record source map mapping
    - After processing all lines, check stack is empty (report error for unclosed IF)
    - Build SourceMap by recording: for each output line N, what was the original line M

    The "currently active" check must AND all stack levels — a nested IF inside an inactive outer IF is always inactive.

    Use pkg/diag.Diagnostic with source.Pos for error positions. Set Pos.File from opts.Filename, Pos.Line from the 1-based original line number, Pos.Col to 1 (directive starts at beginning of meaningful content).
  </action>
